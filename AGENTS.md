# BalCheck.in — Agent Instructions

**balcheck.in** — Bank missed-call balance directory. Astro static site on Cloudflare Pages.
Stack: Astro 4, TypeScript, Tailwind CSS, Cloudflare Pages

## Critical Rules
1. **Never deploy manually** — push to main triggers GitHub Actions
2. **Mobile-first** — rural users on basic smartphones
3. **Hindi language** — all content in Hindi, UI labels in English
4. **Build must pass** before any commit
5. **Data accuracy** — bank numbers must be from official sources. Include source URL in data file comments.

## Commands
```bash
npm run dev       # Dev server (localhost:4321)
npm run build     # Production build
npm run preview   # Preview with Wrangler
```

## Project Structure
```
src/
  pages/
    index.astro              # Main page (directory + search + category tabs)
    how-it-works.astro       # How-to guide page with HowTo + FAQPage schema
    bank/[slug].astro        # 49 bank detail PSEO pages (BankOrCreditUnion + FAQPage + HowTo schema)
    banks/[category].astro    # 6 category listing pages (CollectionPage + ItemList schema)
  layouts/Layout.astro       # HTML layout, meta tags, GA4
  components/SchemaOrg.astro # JSON-LD structured data (homepage)
  data/banks.ts              # Bank missed-call number database (49 banks, 6 categories)
public/
  favicon.svg
  robots.txt                 # Sitemap: https://balcheck.in/sitemap-index.xml
```

## PSEO Pages
- **49 bank pages** at `/bank/[slug]` — each with BankOrCreditUnion, FAQPage, HowTo, BreadcrumbList schema
- **6 category pages** at `/banks/[category]` — CollectionPage + ItemList schema
- **How-to page** at `/how-it-works` — HowTo + FAQPage schema
- Sitemap auto-generated by @astrojs/sitemap at `/sitemap-index.xml`
- Total: 57 pages

## Deployment
Push to `main` → GitHub Actions → Cloudflare Pages. Never use `wrangler deploy` directly.

## Data Sources
Bank missed-call numbers sourced from:
- Official bank websites
- RBI financial education materials
- Public domain banking directories
